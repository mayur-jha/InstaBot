How to Install & Use

  1. 1

    Load the Extension

    Open Chrome and go to chrome://extensions/. Enable "Developer mode" and click "Load unpacked". Select this project folder.

  2. 2

    Get Your API Key

    Sign up at platform.openai.com and create an API key. Save it somewhere safe!

  3. 3

    Configure the Extension

    Click the extension icon in Chrome toolbar, paste your API key, and click "Save".

  4. 4

    Start Automation

    Open Instagram Direct Messages, then click "Start Automation" in the extension. Watch as it processes your conversations!
